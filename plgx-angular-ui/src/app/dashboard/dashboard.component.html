Windows Installation:

                               
 c:\user\downloads\> plgx_cpt.exe -p -i < SERVER_IP > -k < Full path to the certificate file >

Linux CPT Installer
Certificate

Linux Based Installation:

                              
sudo chmod +x plgx_cpt 
sudo ./plgx_cpt -p -i <SERVER_IP> -k <Full path to the certificate file>
Note: Download CPT installer for installation. Use the respective command from terminal for installation using sudo.
